# Day 1: 什么是 AI Agent？

## 概述

AI Agent（AI 代理）是一种能够自主理解任务、规划执行步骤、调用工具并完成目标的智能系统。与传统的 AI 对话不同，Agent 具有**自主行动能力**。

## AI Agent 的核心能力

### 1. 感知 (Perception)
- 理解用户输入
- 读取文件和数据库
- 接收外部信息

### 2. 规划 (Planning)
- 分解复杂任务
- 制定执行步骤
- 反思和调整

### 3. 行动 (Action)
- 调用 API
- 执行代码
- 操作文件系统

### 4. 反思 (Reflection)
- 评估结果
- 错误处理
- 持续改进

## AI Agent vs 传统 AI

| 特性 | 传统 AI | AI Agent |
|------|---------|----------|
| 交互方式 | 问答 | 自主执行 |
| 工具使用 | 无 | 有 |
| 状态管理 | 无状态 | 有状态 |
| 任务类型 | 单轮对话 | 多步骤任务 |

## 主流 AI Agent 框架

1. **OpenClaw** - 本教程使用的 AI 助手框架
2. **Claude Code** - Anthropic 的 AI 编程助手
3. **Cursor** - AI 增强的 IDE
4. **OpenCode** - 开源 AI 编程工具
